Abbreviations

It is used to mark up an abbreviation or acronym:

cdgl was founded in 2020.

Blockquotes

It is used to present content from another source:

I basically wrote the code and the specs and documentation for how the client and server talked to each other.

Tim Berners-Lee

Multiple Code Lines

For multiple lines of code, use the pre element:

I am 
Poulomi SAHA.
I a coding instructor at Codingal.

Keyboard Inputs

To indicate input that is typically entered via the keyboard, use the kbd element:

Use ctrl + p to open the Print dialog box.
